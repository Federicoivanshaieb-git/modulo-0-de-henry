# 🚀 Henry Bootcamp - Módulo 0: Fundamentos de Programación

Este repositorio contiene las bases y los primeros pasos realizados durante el **Prep Course** de Henry. El objetivo principal de este módulo es nivelar conocimientos técnicos y desarrollar el pensamiento lógico necesario para el desarrollo de software profesional.

## 📋 Descripción

En este módulo me enfoqué en dominar los conceptos elementales de la computación y la lógica de programación, estableciendo los cimientos para avanzar hacia el desarrollo Full Stack.

---

## 🛠️ Tecnologías Utilizadas

*   **JavaScript (ES6+):** Sintaxis básica, variables y tipos de datos.
*   **Git & GitHub:** Control de versiones y gestión de repositorios.
*   **HTML5 & CSS3:** Estructura y diseño básico de páginas web.
*   **Markdown:** Documentación técnica.

---

## 🧠 Temas Aprendidos

### 1. Pensamiento Algorítmico
*   Resolución de problemas mediante pasos lógicos.
*   Uso de estructuras de control: `if`, `else`, `switch`.
*   Implementación de bucles: `for`, `while`.

### 2. Fundamentos de JavaScript
*   Manejo de **Variables** (let, const) y **Operadores**.
*   Creación y ejecución de **Funciones**.
*   Introducción a estructuras de datos simples: **Arrays** y **Objetos**.

### 3. Entorno de Desarrollo
*   Instalación y configuración de **VS Code**.
*   Uso de la **Terminal** de comandos.
*   Flujo de trabajo con Git: `clone`, `add`, `commit`, `push`.
